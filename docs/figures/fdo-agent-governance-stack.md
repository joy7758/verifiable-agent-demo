# FDO -> Agent Governance Stack

Reference architecture connecting FAIR Digital Objects with
auditable AI agent execution.

Layers:

FAIR Digital Objects
Global machine-actionable digital object infrastructure.

Verifiable Digital Objects
Objects that contain provenance and execution metadata.

Agent Governance Stack

- Persona (POP)
- Interaction Objects (AIP)
- Governance Checkpoint (Token Governor)
- Execution Trace
- Audit Evidence (ARO)

Purpose:

Enable autonomous agents to produce verifiable,
auditable digital operations inside FDO-based ecosystems.
